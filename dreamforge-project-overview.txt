DreamForge AI is a cutting-edge **AI-based app builder** that takes a user's **voice command or sketch** and turns it into a working web app. It functions as a creative bridge between _imagination and execution_, powered by AI models, multi-agent orchestration, and code generation engines.

It:
- Transforms ideas into code (HTML/JS)
- Understands voice prompts
- Converts rough UI sketches into deployable apps
- Uses Groq AI for faster inference
- Automates frontend + backend setups
